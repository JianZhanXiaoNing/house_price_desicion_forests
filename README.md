# house_price_desicion_forests
本项目与豆包大模型协作，成功构建了预测房屋售价的决策森林模型，可以实现基本的预测功能
在接到任务后，我首先在kaggle上寻找数据集，最终选定了公开可用的数据集——Ames房屋数据集，并下载了相关文件（一个数据说明文件，一个训练csv文件，一个样本csv文件，一个测试csv文件）（另：文件已压缩为zip文件提交）。在选题方面，我一开始A1，A2都做了一些，最终敲定了A1。当然，做的部分A2任务也没有白费功夫，这帮我更加清晰的理解了特征含义。
选定任务和数据集后，我便开始与豆包大模型交互，首先将具有的文件和所要构建的模型任务清晰的投喂给豆包，初步理解了特征含义。
接下来，我便让豆包生成了构建模型的代码。完成最终可执行的代码的过程并不轻松，豆包在其中犯了不少错误，例如运用了seaborn库中没有的barh函数（修改两次后解决）。解决代码问题后，我成功训练了模型，得到了可调用的pkl模型文件。也得到了模型基础性能指标R²、RMSE、MAE的数值。
然后，便是对模型预测错误的分析。我与大模型协作，获得了检验错误并对其分类的代码。修改代码错误后成功运行，得到了错误模式统计报告和分类明细的两个exe文件，并生成了三张可视化直观图。经豆包检验后，得知了哪些错误归因合理、哪些模型误判。
得知错误后，便开始对模型进行优化。根据错误模式，共有四类错误可以优化掉，最终选择最好优化的两种进行了优化。判断可行性后，对模型进行了两次优化。获得了中期模型的pkl文件，和最终模型的pkl文件。并获得了优化前后的性能对比报告及筛选情况。
